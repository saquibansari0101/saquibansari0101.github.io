In July 2022, I joined Infosys as a Digital Specialist Engineer L1. What started as a graduate role quickly evolved into opportunities to lead technical initiatives and own critical systems.

### Major Projects

**EAIGLE Integration**  
Designed and implemented a real-time data integration pipeline streaming trailer, load, and shipment data using Kafka and CDC. This system now processes events for a global supply chain, enabling near-real-time visibility.

**SAP to Kafka Integration**  
Acted as technical lead, architecting a scalable pipeline that ingests 1M+ records from SAP into Kafka, powering multiple downstream consumers.

**Warehouse Dashboard**  
Co-architected a real-time monitoring dashboard integrating decentralized warehouse systems globally. This platform became mission-critical for the client's operations team.

**Data Validation Suite**  
Designed a customizable framework using Python, Pandas, and NumPy that improved data accuracy and consistency across disparate databases.
